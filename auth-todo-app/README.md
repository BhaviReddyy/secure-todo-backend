# Authorization-Based TODO Application

## Tech Stack
Node.js, Express, Supabase, JWT

## Features
- User Signup & Login
- JWT Authentication (1 hour expiry)
- Protected TODO CRUD operations
- User-based authorization

## Setup
1. Clone repo
2. npm install
3. Add .env
4. npm start

## API Endpoints
POST /auth/signup  
POST /auth/login  
POST /todos  
GET /todos  
PUT /todos/:id  
DELETE /todos/:id
